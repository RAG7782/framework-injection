# Ontologia Artesanal — Protocolo de Lapidacao

## Conceito Central
Lapidacao = refinamento iterativo que revela facetas ocultas por passes sucessivos.
Distinto de correcao (consertar erro) e de tempera (testar resistencia).
Lapidacao aprofunda e revela o que ja estava la, mas nao era visivel.

## Estrutura Ontologica

### Camada 1 — Operacoes (7)
```
Enriquecimento ← densificacao de detalhes
Pontos Cegos ← epistemologia negativa
Vieses ← cognicao critica
Alavancagem Pareto ← rarefacao telescopica (20/80 → 5/95 → 1/99)
Expansao ← divergencia criativa orientada
Perguntas Nao-Feitas ← meta-cognicao de 2a ordem
Consolidacao ← integracao preservativa
```

### Camada 2 — Passes (3)
```
Descoberta (divergente) → todas as 7 operacoes
Refinamento (convergente) → operacoes 5, 6, 7 sobre v1
Polimento (divergente→convergente) → operacoes 5, 6, 7 sobre v2
```

### Camada 3 — Modos (3)
```
Automatico → pipeline completo sem pausa
Interativo → operacao por operacao com decisao
Progressivo → pass por pass com decisao
```

### Principios Transversais
```
Anti-bloat: menos elementos, mais inteligencia
Densidade: estrutura > volume, racional > lista
Processamento profundo: CoT/ToT/GoT em cada operacao
```

## Metafora Generativa
O lapidario observa a pedra antes de cortar.
Cada corte revela uma faceta que muda o plano do proximo corte.
Nao existe numero fixo de cortes — o lapidario para quando a gema esta pronta.
Mas a gema sempre pode ser re-lapidada com novas tecnicas ou novos olhos.
